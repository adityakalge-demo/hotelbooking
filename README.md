# hotelbooking
hotelbooking

Swagger UI link:http://localhost:8080/swagger-ui.html

JSON to Create User:{
  "bonusPoints": 0,
  "id": 0,
  "previousBonusPoints": 0,
  "userName": "string"
}


JSON to create Room:{
  "roomId": 0,
  "roomRate": 0,
  "roomStatus": "string",
  "roomType": "string"
}



Note:ID AutoIncremented.

Booking a room request:http://localhost:8080/booking/book?hotelId=1&userId=1
